Как переключить

Одна команда в терминале

Запустите конфигуратор OpenClaw — выберите модель из списка — и все агенты автоматически перейдут на новый «мозг».

terminal · openclaw config
● ready
$ openclaw configure --section model
// откроется меню выбора модели · текущая подсветится зелёным

Память агентов, USER.md, troubleshoot-база — всё сохраняется при смене модели.

⚠️
minimax-free — только для первого теста, чтобы убедиться что установка прошла и бот отвечает. Для любой реальной работы (бизнес-планы, аналитика, контент, автоматизации) сразу ставьте рабочую модель — openai-codex или claude-opus.
